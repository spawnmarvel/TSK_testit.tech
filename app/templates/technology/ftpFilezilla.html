Connect to Linux from Windows with FileZilla SFTP

Are you a recent cloud hosting convert and find yourself struggling to figure out how to best manage the files on your virtual private server (VPS)? Learn that FileZilla provides a user-friendly graphical interface that can securely transfer files to-and-from, as well as move files around within, your VPS.

If you have yet to create an SSH key pair, you can do so by following one of two DigitalOcean tutorials:

Step 1:

For Windows generate SSH key pair


Step 2:

Follow these steps once you have an SSH key pair that you would like to use to connect to your VPS:

When you are done with puttygen.exe, you should have a result like this:


Login to cloud server with putty and alter the authorized_keys file:
sudo nano authorized_keys
And paste in Public key for pasting into OpenSSH authorized_keys file


Then you are ready to configure FileZilla


Step 3:

FileZilla: Edit-> Settings -> SFTP -> browse to the key generated on your machine -> add key


FileZilla: File -> Site manager -> insert ip, port 22, (password is not used here)


Open the new connection you just saved with SSH key, navigate to folders and you are ready for SFTP!
